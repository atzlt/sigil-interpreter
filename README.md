# Sigil

A general-purpose programming language designed with ECS and host interoperability in mind.

## Current status

### Completed

- Expressions and basic statements
- Global functions
- Intrinsics and language items

### In progress

- Additional operator coverage in the lexer/parser
- Additional keyword coverage in the language spec

### Planned

- Closures and function overloading by type
- Structs and interfaces
- Tags and objects
- Queries and filters
- Maybe signals and speculative execution
